السؤال 5 من 20

في المثلث القائم الزاوية، إذا كان طول أحد الأضلاع المجاورة للزاوية القائمة ٨ سم والوتر ١٠ سم، فما هو طول الضلع الآخر؟

طريقة الحل

باستخدام نظرية فيثاغورس للمثلث القائم الزاوية (أ² + ب² = ج²):

  • 'ج' هو الوتر = ١٠ سم.
  • 'أ' هو الضلع المعلوم = ٨ سم.
  • نحتاج لإيجاد 'ب'.
  • ٨² + ب² = ١٠²
  • ٦٤ + ب² = ١٠٠
  • ب² = ١٠٠ - ٦٤ = ٣٦
  • ب = √٣٦ = ٦ سم.
